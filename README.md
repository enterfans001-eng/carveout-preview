# CARVEOUT

CARVEOUTの17LIVE特化ページ向け静的サイトです。

## フォルダ構成

```
HP制作/
├── index.html        # トップページ
├── about.html        # 会社概要
├── services.html     # サポート内容
├── contact.html      # 無料相談フォーム
├── css/style.css     # デザイン
├── js/script.js      # フォーム動作
├── images/           # 画像用
└── assets/           # その他素材用
```

## ページ内容

- CARVEOUT名義の17LIVE特化ファーストビュー
- お知らせ
- 事務所紹介
- 初配信、イベント、データ改善などの支援内容
- 所属までの流れ
- 無料相談フォーム

## 編集する箇所

- `about.html` の会社概要は株式会社エンターファンズの公式情報を反映
- `contact.html` のフォーム送信先
- 必要に応じて `images/` に実際のライバー写真や事務所素材を追加

このサイトは静的HTMLなので、`index.html` をブラウザで開くだけで確認できます。
